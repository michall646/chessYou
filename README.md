# Chess You
It's a simple chess app made with Material You. Written in React Native using expo. Contains Stockfish

## Features
- ♟️functional chess board
- 🤖multiple bots to play agains
- ❓undo, hint
- 🧩puzzles
- 🔎game review 
- 🖌️material you design
- 🎨customizable colors
- 🌙dark mode
